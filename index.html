How to Play

Select an ATP player for each cell that matches both the row and column categories.

1 Click any empty cell to open the player search
2 Search and select a player who fits both categories
3 You have 3 strikes — only wrong guesses cost a strike
4 Correct guesses are free! No penalty for getting it right
5 Each player can only be used once per grid

Rarity Scoring

After filling a cell, you'll see a rarity %. Lower = more unique answer.
